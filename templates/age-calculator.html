Age Calculator

Create an age calculator using HTML, CSS, and JavaScript. This project is perfect for learning how to use external libraries like Luxon and building user-friendly forms with date inputs.

Goal of the Project:

The goal of this project is to teach you how to handle user inputs, manipulate dates, and display results dynamically on a web page. You'll also learn to integrate external libraries and build visually appealing and responsive UI components.

Requirements:

  • Create a form that allows users to input their birthdate using a JavaScript datepicker (avoid default HTML date picker).
  • Use the Luxon library to calculate the exact age in years, months, and days.
  • Display the result dynamically below the form after the user submits it.
  • Implement basic validation to ensure the birthdate is valid.
  • Use CSS to style the form and results for a professional and user-friendly design.
