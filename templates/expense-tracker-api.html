Expense Tracker API

Develop a robust API for tracking and managing expenses. This project integrates user authentication and advanced filtering features to enhance functionality.

Key Features:

  • User Authentication: Sign-up and login functionality with JWT for secure session handling.
  • Expense Management: Create, read, update, and delete expenses.
  • Filtering Options: Filter expenses by past week, past month, last 3 months, or custom date ranges.

Technologies to Use:

  • Backend Framework: Django, Flask, or Express.js
  • Database: SQL (PostgreSQL/MySQL) or NoSQL (MongoDB)
  • Authentication: JSON Web Tokens (JWT)

Expense Categories:

Organize expenses into predefined categories such as:

  • Groceries
  • Leisure
  • Electronics
  • Utilities
  • Clothing
  • Health
  • Others

Project Goals:

Learn to implement RESTful APIs with authentication, database integration, and advanced querying techniques. Build a scalable backend application adhering to best practices.
