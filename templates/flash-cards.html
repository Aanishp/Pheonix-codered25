Flash Cards

Create a flash card app using JavaScript frameworks like React, Angular, or Vue.js. This project will guide you through state management and component-based architecture to build a fun and interactive learning tool.

Goal of the Project:

The goal of this project is to help you learn about how to manage state and create reusable components using JavaScript frameworks. You’ll build a flash card app to help users test their knowledge.

Requirements:

  • Pre-defined flashcards with questions and answers.
  • A progress bar to show the user’s progress.
  • Ability to view one flashcard at a time and flip it to reveal the answer.
  • Simple navigation to cycle through the flashcards.
