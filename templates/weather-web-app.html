Weather Web App

Build a weather web app that fetches and displays weather data for any location entered by the user. The app will provide detailed information like temperature, wind speed, rain likelihood, and more!

Goal of the Project:

The goal of this project is to showcase real-time weather data using an API. You'll learn to fetch data dynamically, manage user input, and present the weather forecast clearly and intuitively. The app should also support refreshing the weather details.

Requirements:

  • Provide an input field for users to enter their desired location (e.g., city name).
  • Display the current weather conditions such as temperature, wind speed, and rain probability.
  • Show a 24-hour weather forecast for the location.
  • Enable users to refresh the weather data at any time.
  • Enhance the user experience with animations or visual transitions when fetching data.
